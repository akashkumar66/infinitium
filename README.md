# Infinitium | Premium Video & Creative Agency Landing Page

A high-converting, premium landing page built with React, Tailwind CSS, and Framer Motion.

## Features
- **Dark Premium Theme**: Futuristic deep navy and indigo aesthetic.
- **Glassmorphism**: Modern frosted-glass UI elements throughout.
- **Framer Motion**: Smooth entry, scroll, and hover animations.
- **11 Sections**: Complete funnel structure (Hero, Trust, Problem/Solution, Services, Comparison, Portfolio, Process, Testimonials, Case Studies, Final CTA, Footer).
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Clean Code**: Component-based architecture with reusable modules.

## Getting Started

1. **Navigate to the project directory**:
   ```bash
   cd c:\Users\aksah\Desktop\Anti\infinitium
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
